# n8n

## Installation

Install **n8n** via npm

```bash
npm i -g n8n
```

## Start

Run below command in **Command Prompt**

```bash
set NODES_EXCLUDE="[]" && set NODE_FUNCTION_ALLOW_EXTERNAL=uuid && set N8N_RESTRICT_FILE_ACCESS_TO="./" && n8n
```

## Import Workflow

Import **Codex Integration Workflow.json** file to your n8n workflows

## Codex CLI

Run below command in **Windows PowerShell** or **Command Prompt**

```bash
npm install -g @openai/codex
```

## Newman (Postman)

Run below command in **Windows PowerShell** or **Command Prompt**

```bash
npm i -g newman
```
